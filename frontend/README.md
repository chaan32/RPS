# RPS Frontend

React + Vite 기반 관리자 대시보드입니다. FastAPI 서버가 `http://127.0.0.1:1122`에서 실행 중이어야 작업자 통계, 사고 로그, 리포트 화면이 정상 동작합니다.

## 실행

```bash
cd /Users/haechan/Desktop/pobiga/ai/ai_project/frontend
npm install
npm run dev
```

기본 개발 서버:

```text
http://localhost:5173
```

## 빌드

```bash
npm run build
```

빌드 산출물인 `dist/`는 Git에 올리지 않습니다.
